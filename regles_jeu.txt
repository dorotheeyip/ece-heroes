Regles du jeu — Projet « ECE Heroes »

Objectif general

Le jeu est un match-3 en console : le joueur doit associer au moins
trois items identiques pour les faire disparaitre et remplir un contrat
d’elimination dans un nombre limite de coups et un temps donne.

Deroulement d’une partie

-   Le joueur commence avec un nombre limite de vies.
-   Une partie contient au moins 3 niveaux de difficulte croissante.
-   Pour gagner la partie, il faut reussir tous les niveaux.
-   Si le joueur perd un niveau, il perd une vie.
-   Si le joueur n’a plus de vie -> defaite.

Menu principal

Le programme doit proposer : 1. Lire les regles du jeu
2. Commencer une nouvelle partie
3. Reprendre une partie
4. Quitter

Plateau de jeu

-   Chaque niveau est un plateau de 25 lignes × 45 colonnes.
-   Les cases sont remplies aleatoirement avec 5 types d’items.
-   Aucune figure speciale ne doit etre presente au debut d’un niveau.

Contrats

Chaque niveau impose : - Un nombre d’items a eliminer (ex : 20 fraises,
50 oignons…) - Un nombre maximal de coups - Un temps limite

Le joueur doit remplir le contrat avant la fin du temps ou des coups.

Figures et eliminations

Certaines formes provoquent des effets speciaux :

1. Ligne horizontale ou verticale de 6 items identiques
  -> Effet : tous les items de ce type disparaissent sur tout le plateau.

2. Croix de 9 items identiques
  -> Effet : tous les items identiques sur la ligne ET la colonne
disparaissent.

3. Carre 4×4 du meme type
  -> Effet : disparition de tous les items contenus dans le carre.

4. Ligne horizontale ou verticale de 4 items identiques
  -> Effet : disparition seulement de la suite.

Chute des items

Apres chaque elimination : 
- Les items au-dessus tombent pour combler les vides. 
- Le haut des colonnes se remplit de nouveaux items aleatoires. 
- Les suppressions se repetent jusqu’a ce que le plateau soit stable.

Deplacement du joueur

Le joueur peut : 
- Se deplacer avec z, q, s, d ou les fleches. 
- Selectionner un item avec espace (un second appui annule). 
- Choisir le voisin a permuter via les touches de deplacement.

Chaque permutation = 1 coup.

Si la permutation cree une figure -> application des regles
d’elimination.

Fin d’un niveau

-   Victoire : contrat rempli -> le joueur passe au niveau suivant.
-   echec : temps ecoule ou coups epuises -> perte d’une vie.

a la fin du niveau : - Le joueur peut continuer ou sauvegarder son
avancee.

Sauvegarde et chargement

-   Au lancement : possibilite de reprendre une partie en saisissant son
    pseudo.

EXTENSIONS 

Nouvelle figures speciales:

1. Diagonale de 4 items identiques
  -> Effet : detruit la diagonale entiere.

2. Carre 2x2 du meme type
  -> Effet : detruit le contour du carre 2x2, detruisant ainsi un carre 4x4.

3. Ligne horizontale ou verticale de 7 items identiques
  -> Effet : fusionne les 7 items alignes en un seul item special du meme type, qui si explose, detruit les 2 diagonales et les lignes verticale et horizontale passant par cette case.

Objets bonus utilisables:

1. Bombe (B)
  -> Effet : applique l'effet de destruction d'un carre 5x5.

2. Fusee horizontale (H)
  -> Effet : applique l'effet de destruction d'une ligne entiere.

3. Fusee verticale (V)
  -> Effet : applique l'effet de destruction d'une colonne entiere.

4. Joker (J)
  -> Effet : remplace un item par un autre item au choix.
